N2O Virtual Node

INTRO

n2o_vnode is an custom instance of n2o_async prorocess service. In the N2O number of such instanses is usually equals to the number of cores.

TOPIC

Virtual node on init performs a subscription with its number in the ring as a name.

PROTOCOL

After supscription all messages that are being sent to events topic are passed as #publish/2 the n2o_async process where n2o_proto:info/3 protocol loop is embedded as handler to that message.

CONFIG

This module may refer to: n2o, n2o_async.
